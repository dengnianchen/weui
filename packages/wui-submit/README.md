# wui-submit

> 表单提交按钮 - 小程序组件

## Install

``` bash
$ min install @yapple/wui-submit
```


## API

### Submit

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`prop-name`           | 描述属性的类型，默认值等         |
|`method-name`         | 描述方法的参数，返回值等         |
